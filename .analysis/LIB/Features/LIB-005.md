# LIB-005: Sound Panel

## Kontekst biznesowy

Panel dzwiekowy (SoundPanel) umozliwia operatorowi natychmiastowe odtwarzanie cartow z siatki przyciskow. Uzywany w RDAirPlay i RDPanel do jingli, efektow dzwiekowych, reklam. Obsluguje wiele stron paneli, konfiguracje przyciskow, drag&drop, hook mode, logowanie odtwarzania.

## Aktorzy

| Aktor | Rola |
|-------|------|
| Operator | Klika przyciski, konfiguruje panele |

## Granica funkcjonalnosci

```
IN SCOPE:
  - RDSoundPanel: widget panelu
  - RDPanelButton: przycisk z odliczaniem, flash
  - RDButtonPanel: kontener przyciskow
  - RDButtonDialog: dialog konfiguracji przycisku
  - Tryby: Full/Hook, setup mode

OUT OF SCOPE:
  - Log playback (LIB-003)
  - Cart dialog (LIB-010)
```

## Reguly biznesowe (Gherkin)

```gherkin
Rule: Przycisk odtwarza przypisanego carta
  Scenario: Klikniecie przycisku
    Given przycisk z przypisanym cartem 100001
    When operator klika
    Then audio gra, przycisk miga i odlicza

Rule: Hook mode odtwarza tylko fragment
  Scenario: Hook mode
    Given przycisk w trybie Hook
    When klikniecie
    Then odtwarzany jest tylko hook segment cuta
```
