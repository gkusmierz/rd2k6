# LIB-011: Platform Abstraction

## Kontekst biznesowy

Warstwa abstrakcji platformowej obejmuje transfer plikow (HTTP/FTP/SFTP), sterowanie GPIO (kernel, LiveWire, Modbus), obsluge CD (playback, ripping, disc lookup), oraz komunikacje z urzadzeniami Axia LiveWire. Wszystkie te komponenty sa platform-specific i wymagaja zamiennikow w klonie.

## Aktorzy

| Aktor | Rola |
|-------|------|
| System | Transferuje pliki, steruje GPIO, komunikuje z hardware |
| Operator | Rippuje CD, szuka metadanych plyt |

## Granica funkcjonalnosci

```
IN SCOPE:
  - RDDownload/RDUpload/RDTransfer: transfer plikow
  - RDLiveWire: LWRP protocol, routing, GPIO
  - RDCdPlayer: odtwarzanie CD
  - RDCdRipper: ripping CD
  - RDDiscLookup/RDCddbLookup/RDMbLookup: disc metadata
  - RDGpio/RDKernelGpio: GPIO
  - RDHash: SHA1 hashing
  - RDSendMail: email
  - RDTty/RDTtyDevice/RDTtyOut: serial TTY

OUT OF SCOPE:
  - Audio engine (artefakt CAE)
  - Matryca sprzetowa (model RDMatrix jest w LIB-001)
```

## Platform Independence

| Funkcja | Rivendell | Klon | Priorytet |
|---------|-----------|------|-----------|
| File transfer | libcurl | [implementator] | MEDIUM |
| CD ripping | cdparanoia | [implementator] | HIGH |
| CD playback | /dev/cdrom ioctl | [implementator] | HIGH |
| CD metadata | libmusicbrainz, CDDB | [implementator] | MEDIUM |
| GPIO kernel | /sys/class/gpio | [implementator] | HIGH |
| LiveWire | LWRP protocol | [implementator] | MEDIUM |
| Serial | TTY (/dev/ttyS*) | [implementator] | MEDIUM |
| SHA1 | OpenSSL libcrypto | [implementator] | LOW |
| Email | sendmail CLI | [implementator] | LOW |
