# RDExportSettingsDialog

## Purpose
Modal dialog for configuring audio export settings. Allows selection of format (PCM, MPEG, Ogg, FLAC, etc.), channels, sample rate, bitrate/quality, normalization level, and autotrim level. Settings are read from and written to an RDSettings object.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDExportSettingsDialog(const QString &caption, QWidget *parent = 0);
```
- `caption` — dialog window title

## Public API / Slots

| Method | Signature | Description |
|--------|-----------|-------------|
| `exec` | `int exec(RDSettings *s, unsigned id = 0)` | Opens the dialog populated with settings from `s`. On OK, writes user selections back to `s`. Optional `id` to show a settings name. |
| `setShowNormalizationLevel` | `void setShowNormalizationLevel(bool state)` | Show/hide the normalization level controls |
| `setShowAutotrimLevel` | `void setShowAutotrimLevel(bool state)` | Show/hide the autotrim level controls |
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `samprateData(const QString &)` — sample rate changed, updates available bitrates
- `bitrateData(const QString &)` — bitrate changed
- `okData()` — saves settings and accepts
- `cancelData()` — cancels dialog

## Key Widgets
- `lib_format_box` (QComboBox) — audio format selector (PCM16/24, MPEG L2/L3, OggVorbis, FLAC)
- `lib_channels_box` (QComboBox) — channel count (1=Mono, 2=Stereo)
- `lib_samprate_box` (QComboBox) — sample rate selector
- `lib_bitrate_box` (QComboBox) — bitrate selector (for lossy formats)
- `lib_quality_spin` (QSpinBox) — quality setting (for Ogg Vorbis)
- `lib_normalization_level_spin` (QSpinBox) — normalization level in dB
- `lib_autotrim_level_spin` (QSpinBox) — autotrim level in dB
- `lib_name_edit` (QLineEdit) — optional settings name display

## Dependencies
- RDSettings (format, channels, sample rate, bitrate, quality, normalization, autotrim)
- RDDialog
