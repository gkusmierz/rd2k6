# RDWaveDataDialog

## Purpose
Modal dialog for viewing and editing audio metadata (wave data) associated with a cart/cut. Displays fields for title, artist, year, usage, song ID, BPM, album, label, client, agency, publisher, composer, conductor, and user-defined data. Also provides access to scheduler codes.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDWaveDataDialog(const QString &caption, QWidget *parent = 0);
```
- `caption` — dialog window title

## Public API / Slots

| Method | Signature | Description |
|--------|-----------|-------------|
| `exec` | `int exec(RDWaveData *data)` | Opens the dialog populated from `data`. On OK, writes edited values back to the RDWaveData object. |
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `schedClickedData()` — opens scheduler codes sub-dialog (RDSchedCodesDialog)
- `okData()` — saves metadata and accepts
- `cancelData()` — cancels dialog

## Key Widgets
- `wave_title_edit` (QLineEdit) — title
- `wave_artist_edit` (QLineEdit) — artist
- `wave_year_edit` (QLineEdit) — year
- `wave_usage_box` (QComboBox) — usage type selector
- `wave_songid_edit` (QLineEdit) — song ID
- `wave_bpm_spin` (QSpinBox) — beats per minute
- `wave_album_edit` (QLineEdit) — album
- `wave_label_edit` (QLineEdit) — record label
- `wave_client_edit` (QLineEdit) — client

- `wave_sched_button` (QPushButton) — open scheduler codes dialog
