# RDPasswd

## Purpose
Modal dialog for changing a password. Requires the user to enter the new password twice for confirmation. Validates that both entries match before accepting.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDPasswd(QString *password, QWidget *parent = 0);
```
- `password` — output: new password string (set only when both entries match and user confirms)

## Public API

| Method | Signature | Description |
|--------|-----------|-------------|
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `okData()` — validates matching passwords and accepts
- `cancelData()` — cancels dialog

## Key Widgets
- `passwd_password_1_edit` (QLineEdit) — first password entry (masked)
- `passwd_password_2_edit` (QLineEdit) — confirmation password entry (masked)

## Dependencies
- RDDialog
