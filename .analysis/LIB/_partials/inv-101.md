# RDEditPanelName

## Purpose
Simple modal dialog for editing a sound panel name. Prompts for a single text string.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDEditPanelName(QString *panelname, QWidget *parent = 0);
```
- `panelname` — in/out: current panel name on entry, edited name on accept

## Public API

| Method | Signature | Description |
|--------|-----------|-------------|
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `okData()` — accepts the entered name
- `cancelData()` — cancels dialog

## Key Widgets
- `panel_name_edit` (QLineEdit) — panel name input field

## Dependencies
- RDDialog
