# RDCartDialog

## Purpose
Modal dialog for browsing and selecting a cart from the Rivendell cart library. Provides filtering by text, group, and scheduler code, with optional cart editor launch and file import.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDCartDialog(QString *filter, QString *group, QString *schedcode,
             const QString &caption, QWidget *parent = 0);
```
- `filter` ‚Äî persistent text filter (in/out, remembered across invocations)
- `group` ‚Äî persistent group filter (in/out)
- `schedcode` ‚Äî persistent scheduler code filter (in/out)
- `caption` ‚Äî dialog window title

## Public API / Slots

| Method | Signature | Description |
|--------|-----------|-------------|
| `exec` | `int exec(int *cartnum, RDCart::Type type, QString *svcname, int svc_quan, const QString &username, const QString &passwd, bool *temp_allowed = NULL)` | Opens the dialog; returns selected cart number via pointer. Filters by cart type (Audio/Macro). Accepts service filter list and user credentials for cart editor access. `temp_allowed` controls whether temporary carts can be created. |
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots (UI Interactions)
- `filterSearchedData()` ‚Äî user clicked Search button
- `filterClearedData()` ‚Äî user clicked Clear filter button
- `groupActivatedData(const QString &group)` ‚Äî group combo changed
- `schedcodeActivatedData(const QString &schedcode)` ‚Äî scheduler code combo changed
- `limitChangedData(int state)` ‚Äî "show results limit" checkbox toggled
- `clickedData(Q3ListViewItem *item)` ‚Äî single click on cart list item
- `doubleClickedData(Q3ListViewItem *, const QPoint &, int)` ‚Äî double click selects and accepts
- `editorData()` ‚Äî launch cart editor for selected cart
- `loadFileData()` ‚Äî import audio file into selected cart
- `okData()` ‚Äî OK button
- `cancelData()` ‚Äî Cancel button

## Key Widgets
- `cart_cart_list` (RDListView) ‚Äî main cart list view
- `cart_filter_edit` (QLineEdit) ‚Äî text filter input
- `cart_group_box` (QComboBox) ‚Äî group filter dropdown
- `cart_schedcode_box` (QComboBox) ‚Äî scheduler code filter dropdown
- `cart_limit_box` (QCheckBox) ‚Äî results limit toggle
- `cart_editor_button` (QPushButton) ‚Äî open cart editor
- `cart_file_button` (QPushButton) ‚Äî import audio file

## State Persistence
Dialog saves/restores filter state (text filter, group, schedcode) between invocations via `LoadState()`/`SaveState()` using a state file.

## Dependencies
- RDCart, RDListView, RDDialog
- RDCart::Type enum (Audio/Macro filtering)
- Progress dialog for long operations
- RDSimplePlayer for audio preview
