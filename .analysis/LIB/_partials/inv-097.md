# RDAddLog

## Purpose
Modal dialog for creating a new log. Collects log name and associated service from the user.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDAddLog(QString *logname, QString *svcname, RDLogFilter::FilterMode mode,
         const QString &caption, QWidget *parent = 0);
```
- `logname` — output: entered log name
- `svcname` — output: selected service name
- `mode` — filter mode controlling which services are shown
- `caption` — dialog window title

## Public API

| Method | Signature | Description |
|--------|-----------|-------------|
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `okData()` — validates and accepts
- `cancelData()` — cancels dialog
- `nameChangedData(const QString &str)` — enables/disables OK button based on whether name is non-empty

## Key Widgets
- `add_name_edit` (QLineEdit) — log name input
- `add_service_box` (QComboBox) — service selection dropdown
- `add_ok_button` (QPushButton) — OK button (disabled when name empty)

## Dependencies
- RDLogFilter::FilterMode, RDDialog
