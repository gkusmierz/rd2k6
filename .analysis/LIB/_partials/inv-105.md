# RDListLogs

## Purpose
Modal dialog for browsing and selecting a log from the system. Displays a filterable list of logs and returns the selected log name.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDListLogs(QString *logname, RDLogFilter::FilterMode mode, QWidget *parent = 0);
```
- `logname` — output: selected log name
- `mode` — filter mode controlling which logs/services are displayed

## Public API

| Method | Signature | Description |
|--------|-----------|-------------|
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `filterChangedData(const QString &where_sql)` — filter widget changed, refreshes list with new SQL WHERE clause
- `doubleClickedData(Q3ListViewItem *, const QPoint &, int)` — double click selects and accepts
- `okButtonData()` — OK button
- `cancelButtonData()` — Cancel button

## Key Widgets
- `list_log_list` (Q3ListView) — log list view
- `list_filter_widget` (RDLogFilter) — integrated filter widget (service/type filtering)

## Dependencies
- RDLogFilter (filter widget with FilterMode)
- Q3ListView
- RDDialog
