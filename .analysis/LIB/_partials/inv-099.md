# RDGetAth

## Purpose
Simple modal dialog for entering an audio threshold value (double). Used for configuring silence detection or trim levels.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDGetAth(double *ath, QWidget *parent = 0);
```
- `ath` — output: entered audio threshold value (double)

## Public API

| Method | Signature | Description |
|--------|-----------|-------------|
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `okData()` — accepts the entered value
- `cancelData()` — cancels dialog

## Key Widgets
- `ath_ath_edit` (QLineEdit) — threshold value input

## Dependencies
- RDDialog
