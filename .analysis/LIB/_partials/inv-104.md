# RDSchedCodesDialog

## Purpose
Modal dialog for selecting scheduler codes to assign to or remove from a cart/cut. Uses a dual-list selector pattern: available codes on one side, selected codes on the other. Manages both "add" and "remove" code lists.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDSchedCodesDialog(QWidget *parent = 0);
```

## Public API / Slots

| Method | Signature | Description |
|--------|-----------|-------------|
| `exec` | `int exec(QStringList *sched_codes, QStringList *remove_codes)` | Opens the dialog. `sched_codes` contains codes to add (in/out), `remove_codes` contains codes to remove (in/out). On OK, both lists are updated with user selections. |
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `okData()` — accepts selections
- `cancelData()` — cancels dialog

## Key Widgets
- `codes_sel` (RDListSelector) — dual-list selector for scheduler codes to add
- `remove_codes_sel` (RDListSelector) — dual-list selector for scheduler codes to remove

## Dependencies
- RDListSelector (dual-list picker widget)
- RDDialog
