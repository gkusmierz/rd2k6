# RDAddCart

## Purpose
Modal dialog for creating a new cart. Collects group, cart type (Audio/Macro), and title from the user.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDAddCart(QString *group, RDCart::Type *type, QString *title,
          const QString &caption, const QString &username,
          RDSystem *system, QWidget *parent = 0);
```
- `group` — output: selected group name
- `type` — output: selected cart type (Audio or Macro)
- `title` — output: entered cart title
- `caption` — dialog window title
- `username` — current user (determines available groups)
- `system` — system configuration reference

## Public API

| Method | Signature | Description |
|--------|-----------|-------------|
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `okData()` — validates and accepts input
- `cancelData()` — cancels dialog

## Key Widgets
- `cart_group_box` (QComboBox) — group selection dropdown
- `cart_type_box` (QComboBox) — cart type selection (Audio/Macro)
- `cart_title_edit` (QLineEdit) — cart title input

## Dependencies
- RDCart::Type, RDSystem, RDDialog
