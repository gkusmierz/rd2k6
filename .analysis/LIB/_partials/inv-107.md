# RDListSvcs

## Purpose
Modal dialog for selecting a service from a list. Shows all available Rivendell services and returns the selected service name.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDListSvcs(const QString &caption, QWidget *parent = 0);
```
- `caption` — dialog window title

## Public API / Slots

| Method | Signature | Description |
|--------|-----------|-------------|
| `exec` | `int exec(QString *svcname)` | Opens the dialog. On OK, writes selected service name to `svcname`. |
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `doubleClickedData(Q3ListBoxItem *item)` — double click selects and accepts
- `okData()` — OK button
- `cancelData()` — Cancel button

## Key Widgets
- `edit_svc_list` (Q3ListBox) — service list

## Dependencies
- RDDialog
