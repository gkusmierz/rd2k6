## Filtrowanie danych (SQL)
Tabela: LOGS. Warunki bazowe:
- TYPE=0 (standard log)
- LOG_EXISTS="Y"
- START_DATE <= today OR null/0000-00-00
- END_DATE >= today OR null/0000-00-00
- + dynamiczny WHERE z RDLogFilter::whereSql()
