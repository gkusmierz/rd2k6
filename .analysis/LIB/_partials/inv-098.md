# RDGetPasswd

## Purpose
Simple modal dialog for prompting the user to enter a password. Returns the entered password via pointer.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDGetPasswd(QString *passwd, QWidget *parent = 0);
```
- `passwd` — output: entered password string

## Public API

| Method | Signature | Description |
|--------|-----------|-------------|
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `okData()` — accepts the entered password
- `cancelData()` — cancels dialog

## Key Widgets
- `pw_password_edit` (QLineEdit) — password input field (masked)

## Dependencies
- RDDialog
