# RDListGroups

## Purpose
Modal dialog for selecting a cart group from a list. Shows only groups accessible to the specified user.

## Inheritance
- **Parent class:** RDDialog (QDialog)
- **Qt meta:** Q_OBJECT

## Constructor
```cpp
RDListGroups(QString *groupname, const QString &username, QWidget *parent = 0);
```
- `groupname` — output: selected group name
- `username` — current user (determines which groups are visible)

## Public API

| Method | Signature | Description |
|--------|-----------|-------------|
| `sizeHint` | `QSize sizeHint() const` | Preferred dialog size |
| `sizePolicy` | `QSizePolicy sizePolicy() const` | Size policy |

## Signals
None.

## Private Slots
- `doubleClickedData(Q3ListViewItem *item, const QPoint &pt, int col)` — double click selects and accepts
- `okData()` — OK button
- `cancelData()` — Cancel button

## Key Widgets
- `group_group_list` (Q3ListView) — group list view

## Dependencies
- RDDialog
- User-based group access control (BuildGroupList filters by username)
