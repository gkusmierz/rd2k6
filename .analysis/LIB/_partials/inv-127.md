## Opis

Reprezentacja uzytkownika systemowego (Linux/UNIX, nie Rivendell). Odczytuje dane z /etc/passwd (getpwnam). Walidacja hasla przez PAM.

## Klasa: RDSystemUser

- **Nie dziedziczy z QObject** — plain C++ class
- **Plik naglowkowy**: lib/rdsystemuser.h
- **Plik implementacji**: lib/rdsystemuser.cpp
- **Zaleznosci**: RDPam (do walidacji hasla)

### Konstruktor

| Sygnatura | Opis |
|-----------|------|
| `RDSystemUser(const QString &username)` | Wczytuje dane uzytkownika z systemu (getpwnam). Jesli uzytkownik nie istnieje, system_exists=false. |

### Metody publiczne

| Metoda | Zwraca | Opis |
|--------|--------|------|
| `username()` | QString | Nazwa uzytkownika |
| `exists()` | bool | Czy uzytkownik istnieje w systemie |
| `uid()` | uid_t | User ID |
| `gid()` | gid_t | Group ID |
| `fullName()` | QString | Pelna nazwa (z pola GECOS) |
| `homeDirectory()` | QString | Katalog domowy |
| `shell()` | QString | Domyslna powloka |
| `validatePassword(const QString &pwd)` | bool | Waliduje haslo przez PAM (serwis "login") |

### Pola prywatne

| Pole | Typ | Opis |
|------|-----|------|
| system_username | QString | Nazwa uzytkownika |
| system_exists | bool | Czy istnieje |
| system_uid | uid_t | UID |
| system_gid | gid_t | GID |
| system_full_name | QString | Pelna nazwa |
| system_home_directory | QString | Katalog domowy |
| system_shell | QString | Powloka |
| system_password | QString | (nieuzywane w kodzie) |

## Zachowanie kluczowe

- Konstruktor wywoluje getpwnam() — odczyt z /etc/passwd (lub NSS)
- Jesli uzytkownik nie istnieje: exists()=false, reszta pol nieokreslona
- **Uwaga**: system_exists nigdy nie jest ustawiane na true w konstruktorze (prawdopodobny bug — getpwnam sukces nie ustawia system_exists=true)
- validatePassword() deleguje do RDPam::authenticate() z serwisem "login"
- system_password pole istnieje ale nie jest uzywane (hasla walidowane przez PAM, nie bezposrednio)
