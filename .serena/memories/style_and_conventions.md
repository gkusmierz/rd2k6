# Code Style and Conventions

- C++ with Qt framework
- Class naming: PascalCase with RD prefix (RDCart, RDCut, RDStation)
- Method naming: camelCase
- Member variables: prefixed (varies)
- File naming: lowercase, often matching class name (rdcart.h, rdcart.cpp)
- Headers use include guards
- Qt signals/slots mechanism extensively used
- SQL queries embedded in C++ code (no ORM)
- See CODINGSTYLE file for detailed conventions
